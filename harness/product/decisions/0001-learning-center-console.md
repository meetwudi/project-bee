# Decision 0001: Learning Center Console

## Status

Accepted.

## Date

2026-06-10.

## Context

The product direction is to solve the learning problem with a center console for learning. The console should handle lots of information and be intelligent enough to build a learning loop: learn what the user tends to learn, discover what to learn next, and help make sense of it.

## Decision

This repo will treat the learning center console as the core product direction.

## Consequences

- Product work should focus on adaptive learning loops, information synthesis, and learning discovery.
- Requirements should avoid generic content-dashboard behavior unless it supports the learning loop.
- Unknown product facts should stay explicit until supplied by a human or observed in the repo.
